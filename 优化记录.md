## 2026-03-24

### 本次优化文件
- `semi_sup/scripts/generate_ensemble_pseudo_annotations.py`

### 本次优化内容
1. **IoU 聚类策略升级**
   - 从“匹配第一个满足阈值的簇”改为“匹配 IoU 最高的簇（best match）”。
2. **WBF 评分更鲁棒**
   - 由简单分数融合升级为 `top + p75 + mean` 的组合评分。
3. **支持模型可靠性权重**
   - 新增 `--model_weights_json` 参数，可按模型质量进行加权。
4. **共识因子增强**
   - 在 spread 惩罚 + 模型数奖励基础上，加入簇大小轻量奖励。
5. **新增 Soft-NMS 后处理**
   - 对同图同类结果进行高重叠软抑制，减少重复框。
6. **新增质量过滤**
   - 加入最小面积与最大长宽比约束，过滤异常伪框。
7. **脚本参数化增强**
   - 关键阈值全部 CLI 化，便于进行消融与网格搜索。
8. **输出格式增强**
   - 输出注释补齐 `id / area / iscrowd`，COCO 兼容性更好。

### 与之前版本的不同
- **聚类阶段**：从“首个命中”改为“最佳匹配命中”，减少错簇。
- **融合评分**：从基础融合分数升级为更稳健的统计组合分数。
- **后处理链路**：新增 Soft-NMS 与质量过滤，之前无该步骤。
- **可调性**：新增多个 CLI 参数，之前参数多为脚本内常量。
- **输出规范**：新增 COCO 关键字段补齐，之前字段不完整。

### 预期收益
- 提升伪标签质量（减少重叠冗余与噪声框）。
- 提升半监督训练稳定性与最终检测准确率（mAP）。
- 提高实验可复现性与参数调优效率。

## 2026-03-25

### 本次优化文件
- `semi_sup/scripts/sweep_ensemble_params.py`（新建）

### 本次优化内容
1. **新增四参数网格扫描脚本**
   - 自动扫描 `tau_f / theta / min_votes / soft_nms_iou` 组合。
2. **每组参数独立输出伪标注文件**
   - 防止覆盖，便于回溯与复现实验。
3. **自动记录扫描结果到 CSV**
   - 每组记录生成状态、输出路径、伪标注数量、日志尾部。
4. **支持可选评估命令模板**
   - 可在每组伪标注生成后自动执行评估，并记录返回码与日志。
5. **增加关键优化注释**
   - 在脚本中标注了参数扫描、输出命名、结果记录等核心逻辑说明。

### 与之前版本的不同
- 之前需要手动改参数并重复运行；现在可一键批量扫描。
- 之前没有标准化结果汇总；现在会统一输出 CSV 便于筛选最优参数。
- 之前参数-结果映射追踪成本高；现在每组输出文件名与参数强绑定。

### 预期收益
- 显著降低调参成本，提高找到最优参数组合的效率。
- 更快定位高质量伪标注配置，从而间接提升最终识别准确率。
